Welcome to the repository for **Akina Wa**, a WhatsApp bot inspired by neko aesthetics. This project is currently in **Beta version**, under active development, and not yet complete. It may contain bugs or unfinished features. Contributions and feedback are welcome to help improve it.

## Description
Akina Wa is a WhatsApp bot designed to handle commands with a neko-themed flair, including sending stickers, memes, and informational responses. Built with Node.js and whatsapp-web.js, it aims to provide an engaging user experience. Current version: Beta v0.1.0.

## Installation
To set up and run the bot locally (requires Node.js):

1. Clone the repository:  
   ```bash
   git clone https://github.com/Dev-FelixOfc/AkinaWa-Bot.git
   ```
2. Install dependencies:  
   ```bash
   npm install
   ```
3. Configure your WhatsApp account by scanning the QR code generated during setup.
4. Start the bot:  
   ```bash
   npm start
   ```

**Beta Note**: The bot may require additional configuration or troubleshooting due to its developmental stage.

## Usage
Add the bot to your WhatsApp contacts or group, then interact using prefixed commands. Examples include:

- `#miau`: Receives a neko-style greeting.
- `#sticker`: Converts an attached image into a sticker.
- `#info`: Provides curious facts about topics like cats or anime.
- `#juego`: Initiates a simple guessing game.
- `#menu`: Displays a list of available commands (in development).

Responses incorporate neko elements for a themed interaction.

## Features
Akina Wa includes the following specifications, with many in Beta or planned:

- **Multi-Language Support**: English and Spanish (primary), with Japanese support upcoming.
- **Custom Commands**: Ability to create group-specific commands (Beta feature).
- **API Integrations**: Connects to external APIs for weather, news, and meme generation.
- **Neko Mode**: Random responses with neko-inspired phrases.
- **Anti-Spam Security**: Cooldown periods to prevent abuse.
- **Logging**: Interaction logs for debugging purposes.
- **Over-the-Air Updates**: Planned automatic updates controlled by the owner.
- **Visual Themes**: Sends neko-themed images.
- **Voice Integration**: Experimental transcription of voice messages.
- **Compatibility**: Works in both group chats and private messages, with admin permissions.
- **Resource Efficiency**: Optimized for low CPU and RAM usage, suitable for small VPS.
- **Dependencies**: whatsapp-web.js, qrcode-terminal, and image processing libraries.
- **Beta Limitations**: Does not support long videos; occasional disconnections may occur.
- **Future Features**: Neko RPG mode, Telegram integration, and AI-enhanced commands.
